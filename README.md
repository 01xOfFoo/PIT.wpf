#PIT
###Digging out issues

PIT is a uncomplicated C#, WPF, MVVM, RESTful-to-be & what not, bug and issue tracking system which supports the software development lifecycle.

##Current state

PIT is currently in a pre-alpha state and not fit for purpose.
